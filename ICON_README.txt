Home Assistant Add-on Icon
===========================

You should create an icon.png file (108x108 pixels) for your add-on.

Suggested design:
- Background: Dark blue or gray
- Icon: Stylized representation of C-Bus + Home Assistant
- Could use: Light bulb icon, network/bridge symbol, or C-Bus logo

Tools to create icons:
- GIMP (free, open-source)
- Inkscape (free, vector graphics)
- Online tools like Canva

Once created, save as:
  icon.png (108x108 pixels, PNG format)

This icon will appear in the Home Assistant add-on store and add-on details page.

If no icon is provided, Home Assistant will use a default placeholder icon.
